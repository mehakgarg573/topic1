Activity-1

  • What is an IoT Application?

    In layman language, an IoT Application is a connection of objects to each other and the connection of those objects to human beings (through internet). It includes: -

    • Ai and machine learning algorithms.
    • Connecting sensor data to cloud.
    • An easy access with dashboard.
    • Providing the real-time data for many different purposes.
    • Analyzing the data.

  • What are the main features needed?

    The following features are required in an IoT Application: -

    • Good quality sensors
    • Cyber security
    • Real time data
    • Connectivity
    • Integration
    • Automation

  • What are some examples?

    Here are a few exaples of an IoT Application: -

    • Healthcare systems
    • Smart escalators
    • Automatic/smart doors
    • Automatic/smart taps
    • Alexa /Google assistant
    • Pacemakers
    • Smart watch, Smart phone
    • Laptop

Activity-2

  • What does smart mean?

    The word smart can be explained in the following terms: -

    • Objects that are equipped with positioning.
    • Included with communication technologies.
    • Efficient and smart use of economic resources
    • Providing the real-time data for many different purposes.
    • Integrated into a network

  • What are the main features needed?

    The features required in a smart building are: -

    • IoT sensors
    • AI
    • Controlled appliances
    • Energy management
    • Keyless entry
    • Real time alerts
    • Security and surveillance

  • What are some smart-building scenarios?

    This is a scenario of a smart corporate building inspired from The Edge Building in Amsterdam.: -

    • Automatic doors (easy access)
    • Smart pantry
    • Lifts
    • Facial recognition (attendance of employees)
    • Automatic lightning (to save electricity)
    • Automatic HVAC
    • Smoke detectors and automatic water sprinklers (to prevent fire breakout)
    • Automatic smart watering system
    • Smart parking system
